# short_url_generator
介紹

將長網址縮短



專案畫面
![學期2-3 A11 短網址產生器](https://user-images.githubusercontent.com/98015628/159498112-83ad77a1-244d-40f6-9b80-d9458c6ca3c3.png)


產品功能
1. 輸入長網址可得到5碼英數亂碼作為縮網址
2. 將縮網址輸入至瀏覽器網址列可連線至原始網站(長網址)

開發工具
1. Node.js v12 以上
2. Nodemon v2.0.15
3. Express v4.16.4
4. Express-handlebars v3.0.0
5. Bootstrap 5
6. MongoDB
7. Mongoose v6.2.7
